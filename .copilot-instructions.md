## Game Instructions

 - *The snake must move by W A S D*
 - W key is to move forward
 - A key is to move left
 - S key is to move back
 - D key is to move right
 - *Before the game starts a button appears on the screen saying begin game*
 - When the begin game button is pressed the game starts
 - *A singular red apple with pop up when the game starts*
 - The snake has to be on the same space as the apple for the apple to be registered as eaten before disappearing
 - *When the snake has touched the apple on the same space it disappears and a new apple spawns somewhere else on the grid*
 - When the snake touches the apple on the same space the apple is on an eating sound happens for three seconds
 - The snake has to keep eating the apples to grow longer until the snake fills up the whole gird
 - When the snake fills up the whole grid by eating apples the game freezes and turns tinted and a trophy icon is displayed
 - A button will appear when the trophy icon is displayed saying restart
 - The snake game restarts

 ### Grid layout and instructions
 - *The grid is 15x17 grid*
 - The grid has a checker pattern consisting of light green and dark green
 - The border of the checkered grid is coloured black
 - If the snakes touches the border of the grid head on it dies and the game has to restart

#### The snake appearance
 - The snake is one 1 block wide and 2 blocks in length
 - The first block in length of the snake has a black circle indicating the main movement
 - The second block and the blocks later on will follow the previous movement of the first block in length 
 - The blocks of the snake are coloured blue 
 - *The snake has to touch the apple to grow an extra block in length*
 - The extra block will be placed behind the first snake block and follow the first snake block previous movements

 ## Game layout
 - A counter starting from zero is displayed on the screen
 - When the snake touches an apple the counter increases by one
 - The counter will keep increasing based on the amount of apples touched until the game freezes and turns tinted
